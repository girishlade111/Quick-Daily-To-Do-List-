# Quick-Daily-To-Do-List-
Quick Daily To-Do List
**Quick Daily To-Do List**  
**Description**: A minimalist task manager with drag-and-drop prioritization.  
**Features**:  
- Add/delete tasks.  
- Local storage.
- Dark/light mode.  
**Tech**: Flutter (cross-platform) or Kotlin/Swift (native).  
**No Auth Needed**: Data stays on-device.
